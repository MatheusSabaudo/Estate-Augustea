# Estate Augustea

Sito demo realizzato con Next.js per presentare un concept editoriale dedicato a eventi culturali in location archeologiche italiane.

## Nota sul progetto

Questo repository rappresenta un prototipo frontend.
Le pagine `biglietti` e `contatti` simulano il comportamento di un flusso reale, ma non elaborano pagamenti e non inviano dati a un backend.

## Stack

- Next.js
- React
- CSS Modules

## Avvio locale

```bash
npm install
npm run dev
```

## Script disponibili

```bash
npm run dev
npm run build
npm run start
npm run lint
```
